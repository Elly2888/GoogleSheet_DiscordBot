## 注意事項
在Discord開發者頁面選擇Bot後
在TOKEN碼下面有個Privileged Gateway Intents
這裡三個選項都要打開

## 檔案清單

### discordbot.py
主程式

### requirements.txt
外部庫清單

### Procfile
Heroku執行指令

### runtime.txt
Heroku環境設定(Python版本)

### app.json
APP資訊

## 環境變數
### TOKEN
機器人TOKEN

### APIKEY
Google API key

### SHEET_ID
Google試算表ID

### REPLY_SHEET
回覆指令Google試算表單名

### ROLE_SHEET
身分組領取Google試算表單名

### GUILD_ID
伺服器ID

### ROLE_MESSAGE_ID
身分組領取訊息ID
